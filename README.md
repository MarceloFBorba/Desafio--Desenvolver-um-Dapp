# Pixels
Pixels é um Dapp aonde podemos criar e trocar suas próprias artes em pixel 8x8.
você tem o poder de:
    criar qualquer pixel art
    Defina um preço e ganhar cUSD. Você pode alterar o preço a qualquer momento.
    Compre qualquer pixel art que desejar. Você paga uma parte para o artista (10%), fica com o proprietário.

# Você deve instalar o npm

npm install

# Start

npm run dev

# Build

npm run build 

# Como usar:
1. Instale o [CeloExtensionWallet](https://chrome.google.com/webstore/detail/celoextensionwallet/kkilomkmpmkbdnfelcpgckmpcaemjcdh?hl=en) da google store.
2. Crie uma carteira.
3. Vá para [https://celo.org/developers/faucet](https://celo.org/developers/faucet) e obtenha os tokens para o testnet alfajores.
4. Mude para o testnet alfajores no CeloExtensionWallet.
